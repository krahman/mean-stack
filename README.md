## MEAN Stack Web Apps

### Prerequisites
* Install Node.js (if you haven't done so)
* Install mongodb (if you haven't done so)

### To get started
* Clone this repository
```bash
$ git clone git@github.com:krahman/mean-stack.git
```
* Goto project directory
```bash
$ cd mean-stack
```
* Install required node packages
```bash
$ npm install
```
* Install required web components
```bash
$ bower install
```
* Run application
```bash
$ node server.js
```

### Quick run the application
* Use npm
```bash
$ npm start
```

### Run karma test
```bash
$ npm test
```
* Or test single run
```bash
$ npm run test-single-run
```

### Run Protractor e2e testing
```bash
$ npm run protractor
```
